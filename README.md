**THIS IS A REUPLOAD OF SHADOWS AMONG US 3D PUBLIC MOD MENU! THIS IS HERE FOR REUPLOADING PURPOSES, I DID NOT MAKE THIS!**

This is Shadows Among Us 3D mod menu that was released a while back. He has since removed it, but I have it so I decided to put it back up.


**TO USE THIS**: You have to install [MelonLoader](https://melon-loader.com/) into Among Us 3D. You will then have to launch the game once to write all of the neccessary files. After the Title Screen opens, you can close the game. Afterwards, Move the **ShadowsPublicMenu.dll** file into the **Mods** folder located into the directory of Among Us 3D.

Happy modding!
